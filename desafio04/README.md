# DESAFIO

Crie um programa de consulta de CPF utilizando uma gem específica para verificar se o CPF informado pelo usuário é válido ou não.

## Instalação das gems

Antes de tudo, certifique-se de ter o `bundler` instalado em sua máquina. Caso já tenha instalado, clone o repositório, abra-o em um terminal e execute o comando:

~~~ruby
bundle install
~~~

## Executando o desafio

Após instalar as gems, execute o comando no terminal:

~~~ruby
ruby main.rb
~~~
